
Single Page Resume Website (HTML Only)

This project is a clean and simple single-page resume website built using pure HTML. The purpose of this project is to demonstrate proper usage of semantic HTML tags such as headings, paragraphs, lists, tables, and sections without using any CSS.

The resume includes structured sections like Header, About Me, Skills, Experience, Projects, Education, and Contact Details. The focus of this assignment is clean structure, readability, and correct HTML layout.

This project is ideal for beginners who want to practice HTML fundamentals and understand how to organize content properly in a webpage.
